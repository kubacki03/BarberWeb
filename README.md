## Funkcje

## Technologie
-Java
-Spring Web
-Spring Security
-Thymealeaf
-Maven
-JPA/Hibernate
-HTML
-TAILWINDCSS

## Instalacja
1)Sklonuj repozytorium
  git clone https://github.com/kubacki03/BarberWeb.git
2)Przejdz do folderu z projektem
  cd testSecurity
3)Uzupelnij plik application.properties
4)Skompiluj projekt
  ./mvnw clean install
5)Uruchom projekt
  ./mvnw spring-boot:run
